CSS Grid Layout

What is the Grid Layout?

Grid Layout is a new layout model for CSS that has powerful abilities to control the sizing and positioning of boxes and their contents.

Grid vs Flexible Box

Flexible Box Layout is single-axis–oriented. Exemplary Flex Layout Example

Grid Layout is optimized for 2-dimensional layouts. These are layouts in which alignment of content is desired in both dimensions. Exemplary Grid Layout Example

Browser Support

Grid Elements

  • Grid container
  • Grid items

Example

<div class="grid-container">
<div class="grid-item">1</div>
<div class="grid-item">2</div>
<div class="grid-item">3</div>
</div>

Display Property


Example 1

.grid-container {
	display: grid;
}

Example 2

.grid-container {
	display: inline-grid;
}

Grid Layout Concepts

Other Grid Layout concepts are grid columns, grid rows, grid gaps, grid cells, grid areas and grid lines.

Grid Columns

The grid columns are vertical lines of grid items.

Grid Rows

The grid rows are horizontal lines of grid items.

Grid Gaps

The grid gaps are spaces between each column or row.

Grid Cell

The grid cell is the intersection of a grid row and a grid column.

Grid Area

The grid area is the logical space used to lay out one or more grid items. The grid area consists of one or more adjacent grid cells.

Grid Lines

The lines between grid columns are called column lines. The lines between grid rows are called row lines.

Grid Lines

A grid item references the grid lines to determine its position within the grid using the grid-placement properties.

Examples

.item1 {
	grid-column-start: 1;
	grid-column-end: 3;
}
.item1 {
	grid-row-start: 1;
	grid-row-end: 3;
}

Grid Container’s properties

The grid container has several CSS properties.

Grid-template-columns property

defines the number of columns in grid layout and can define the width of each column.

Examples

.grid-container {
	display: grid;
	grid-template-columns: auto auto auto auto;			
}
.grid-container {
	display: grid;
	grid-template-columns: 80px 200px auto 40px;
}

Grid-template-rows property

defines the height of each row.


Example

.grid-container {
	display: grid;
	grid-template-rows: 80px 200px;									
}

Justify-content property

allows to align the whole grid inside the container and has following values: start , end, center, space-evenly, space-between and space-around.

Align-content property

allows to vertically align the whole grid inside the container and has the same values like the justify-content property.

Grid Item’s properties

Grid item has several CSS properties.

Grid-column property

defines on which column(s) to place an item. With this property, we define where the item will start, and where the item will end.

Examples

.item1 {
	grid-column: 1 / 5;
}

.item1 {
	grid-column: 1 / span 3;
}

Grid-row property

defines on which row to place an item.


Examples

.item1 {
	grid-row: 1 / 4;
}

.item1 {
	grid-row: 1 / span 2;
}

Grid-area property

can be used as a shorthand property for grid-row-start, grid-column-start, grid-row-end and grid-column-end properties.

Examples

.item1 {
	grid-area: 1 / 2 / 5 / 6;
}								

.item1 {
	grid-area: 2 / 1 / span 2 / span 3;
}

Naming Grid Items

The grid-area property can also be used to assign names to grid items. Named grid items can be referred to by the grid-template-areas property of the grid container.

Example

.item1 {
	grid-area: myArea;
}
.grid-container {
	grid-template-areas: 'myArea myArea myArea myArea myArea';
}

Order of the Items

The Grid Layout allows us to position the items anywhere we like. We can re-arrange the order for certain screen sizes, by using media queries.

Example

.item1 { grid-area: 1 / 3 / 2 / 4; }
.item2 { grid-area: 2 / 3 / 3 / 4; }
.item3 { grid-area: 1 / 1 / 2 / 2; }
									
@media only screen and (max-width: 500px) {
.item1 { grid-area: 1 / span 3 / 2 / 4; }
.item2 { grid-area: 3 / 3 / 4 / 4; }
.item3 { grid-area: 2 / 1 / 3 / 2; }
}
